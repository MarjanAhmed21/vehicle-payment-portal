# Vehicle Payment Portal

A Laravel + Vue.js payment portal demonstrating REST API development and frontend integration.

## Tech Stack
- PHP 8.2
- Laravel 12
- Vue.js 3
- Vite
- Axios
- Tailwind CSS

## Features
- REST API for payments
- Vue frontend consuming Laravel APIs
- Database migrations and Eloquent models

## Setup
```bash
composer install
npm install
php artisan key:generate
php artisan migrate
npm run dev
php artisan serve
